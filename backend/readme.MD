# backend
lancer le serveur : `nodemon server`
# Les routes
## Post - Créer un compte.
http://localhost:3000/api/auth/signup
## Post - Se Logger.
http://localhost:3000/api/auth/login
## Post - Créer une nouvelle sauce.
http://localhost:3000/api/sauces
## Get - Afficher toutes les sauces.
http://localhost:3000/api/sauces
## Get - Afficher une sauce grâce à son ID.
http://localhost:3000/api/sauces/:id
## Put - Modifier une sauce grâce à son ID.
http://localhost:3000/api/sauces/:id
## Delete - Supprimer une sauce grâce à son ID.
http://localhost:3000/api/sauces/:id
